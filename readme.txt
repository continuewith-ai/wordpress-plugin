Add the ContinueWith AI handoff widget to your WordPress site.

== Description ==

ContinueWith lets visitors continue reading your page in ChatGPT, Claude, Gemini, Grok, Perplexity, or Mistral with one click.

This plugin injects the official ContinueWith widget script on public pages. Configure your public site key under Settings → ContinueWith.

* No chatbot hosting on your server
* Handoff analytics in the ContinueWith dashboard
* Works with any theme

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/continuewith/` or clone from GitHub.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings → ContinueWith and paste your public site key from continuewith.ai/dashboard.

== Frequently Asked Questions ==

= Where do I get a site key? =

Sign up at https://continuewith.ai/dashboard and copy the public key from your site's Install tab.

= Can I use a snippet instead of this plugin? =

Yes. See https://continuewith.ai/docs/install#wordpress
